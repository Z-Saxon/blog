Microsoft PowerToys
更新于 2020-07-26

原文：https://github.com/microsoft/PowerToys

概述

Microsoft PowerToys 是一组实用程序，用于为电力用户调整和简化其 Windows 体验，从而提高工作效率。受到Windows 95 时代 PowerToys 项目的启发，此重新启动为电源用户提供从 Windows 10 外壳中挤出更多效率并针对单个工作流自定义它的方法。Windows 95 PowerToys的概述可以在这里找到。

生成状态

当前PowerToys实用程序

FancyZones

FancyZone 是一个窗口管理器，可以轻松创建复杂的窗口布局，并快速将窗口定位到这些布局中。

File Explorer Add-ons (Preview Panes)

File Explorer Add-ons (Preview Panes)现在仅限于文件资源管理器的预览窗格添加。预览窗格是文件资源管理器中的现有功能。要启用它，只需单击功能区中的”查看”选项卡，然后单击”预览窗格”。

PowerToys 现在可预览两种类型的文件：标记 （.md） 和 SVG （.svg）

Image Resizer

Image ResizerWindows 外壳扩展，用于快速调整映像大小。只需从文件资源管理器中右键单击，即可立即调整一个或多个图像的大小。此代码基于bricelam Image Resizer。

Keyboard Manager

Keyboard Manager允许您通过重新映射键和创建自己的键盘快捷方式来自定义键盘，从而提高工作效率。此 PowerToy 需要 Windows 10 1903（版本 18362）或更晚版本。

PowerRenname

PowerRename 是用于使用搜索和替换或正则表达式进行高级批量重命名的 Windows Shell 扩展。PowerRename 允许简单的搜索和替换或更高级的正则表达式匹配。在键入搜索和替换输入字段时，预览区域将显示项目将重命名为哪些内容。然后，PowerRename 调用 Windows 资源管理器文件操作引擎以执行重命名。这样做的好处是允许在 PowerRename 退出后撤消重命名操作。此代码基于克里斯·戴维斯的智能名称。

PowerToys Run

PowerToys Run是 PowerToys 中的新玩具，它可以帮助您使用简单的 alt 空间立即搜索和启动应用并开始键入！它是开源的，模块化的其他插件。窗口沃克现在也在里面！此 PowerToy 需要 Windows 10 1903（版本 18362）或更晚版本。

Shotcut Guide

当用户将 Windows键向下按 1 秒多显示，并显示桌面当前状态的可用快捷方式时，将显示 Windows 键快捷方式指南。
